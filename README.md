# Python_FINAL
Módulo 10, Lección 4 - Graduación.

Hecho por Alessandro Izquierdo.
Proyecto de graduación de Python: Medidas de Prevención.
Utiliza este pequeño programa para que no te inmuten los estragos del calentamiento global.

Hoy en día, y especialmente en estas épocas veraniegas, el calentamiento global ha sido un factor de gran prominencia a escala global, causando múltiples desastres tales como incendios forestales, sequías y olas de calor.
Lo que busca mi proyecto es proponer modalidades en las que un individuo o un grupo de personas pueden prevenir y/o gestionar riesgos relacionados a los estragos que puede generar el calentamiento global dependiendo de sus condiciones actuales.
La manera en que se realiza esto es mediante el planteamiento de preguntas que respectan a las condiciones de la zona en la que vive el usuario, que este puede responder con un "sí", un "no", o decir que está indeciso.
Lo que este programa busca es recopilar los datos para generar una o múltiples alternativas de solución para afrontar los cambios negativos que puede traer el calentamiento global.

Para utilizar este proyecto, todo lo que se tiene que hacer es instalar este repositorio o copiar/transcribir el código a Microsoft Visual Studio Code, ejecutarlo, revisar la terminal y copiar el enlace que se proporcionará.
